# Phase 8：生态和扩展

目标是让 Specflow 成为可扩展的持续编码 workflow 平台。

## 范围

- 可插拔 agent
- 可插拔 verifier
- 可插拔 workflow 模板
- 自定义节点
- 自定义 artifact 类型
- 可复用 workflow pack
- 团队内部模板
- 未来社区模板
- 高级 graph editing

## 示例 workflow

- 前端功能开发 workflow
- 后端 API 开发 workflow
- bugfix workflow
- refactor workflow
- UI from screenshot workflow
- test generation workflow
- documentation update workflow

## 最终形态

Specflow 不只是一个固定流程工具，而是一个持续编码 workflow 平台。
